Библиотеки анимации:
берут на себя расчёт анимации;
могут автоматически выбирать более эффективный для данной анимации и/или данной аппаратуры способ анимации (вынести ли анимируемый элемент на отдельный GPU-слой, какое именно стилевое свойство анимировать — left или transform, анимировать через CSS или JavaScript...);
берут на себя вопросы кроссбраузерности.

GSAP (GreenSock Animation Platform) https://greensock.com/gsap/
библиотека JavaScript-анимации
100Kb
хорошая документация
перенесён с платформы Flash
состоит из нескольких модулей — Tween (анимация), Timeline (управление воспроизведением), EasePack (пакет формул переходов)
анимации до 20 раз быстрее чем у jQuery
большое количество преднастроенных анимаций
анимации можно собирать в цепочки
включает физический 2D-движок
Internet Explorer поддерживается от 6 версии
пример:
    TweenMax.to(elements, 1, { opacity: 0.5 });
или
    var tl = new TimelineMax();
    tl
      .to(element, 1, { opacity: 0.5 })
      .to(element, 1, { opacity: 1 });
легко интегрируется с jQuery:
  $elements.animate({ opacity: 0.5 });
ниша: задачи с точным контролем времени (например: пауза+возобновление, замедление), контролем движения (дорожки, кривая Безье); разработка игр, но не UI

Velocity.js
библиотека JavaScript-анимации
7Kb
очень высокая скорость, лучше чем у jQuery
совместима по синтаксису с jQuery, можно просто заменить вызовы $(xxx).animate(yyy) на $(xxx).velocity(yyy)
анимирует цвета, трансформации, скроллинг
анимации можно собирать в цепочки
поддерживает SVG
Internet Explorer поддерживается от 8 версии
пример:
    Velocity(elements, { opacity: 0.5 }, 1000);
    Velocity(elements, { opacity: 1 }, 1000);
легко интегрируется с jQuery:
    $elements
      .velocity({ opacity: 0.5 })
      .velocity({ opacity: 1 }, 1000);
позволяет легко анимировать "назад", к предыдущим значениям:
    $elements.velocity("reverse");
умеет работать с физикой, а то есть моделировать естественное движение:
    Velocity(element, { left: 500 }, [ 500, 20 ]); // 500 натяжение, 20 трение
умеет работать со скроллингом:
    $element.velocity("scroll", 1000); // скроллировать к элементу $element
ниша: разработка UI
сайт, демки

библиотека CSS-анимации
позиционируется как быстрая некроссбраузерная замена jQuery (IE 10+ и современные браузеры)
модульная структура — можно загрузить версию только с нужными возможностями
базовая версия — 9kb
синтаксис аналогичен jQuery

Web Animations API
предоставляет JavaScript-API встроенного движка CSS-анимации браузера, которое позволит скриптам и библиотекам повысить производительность и гибкость;
позволяет анимации выполняться в отдельном потоке выполнения CPU, повышая тем самым плавность анимации и независимость от нагрузки на браузер;
поддерживает траектории движения;
обеспечивает вызов коллбэков после проигрывания анимации;
поддерживает вложенные и последовательные анимации;
позволяет останавливать, запускать, искать, проигрывать в обратном порядке, замедлять или ускорять воспроизведение анимации;
разрабатывается, уже есть в Chrome 36+, для остальных браузеров есть полифилл;
