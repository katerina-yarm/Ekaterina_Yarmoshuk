Достоинства Canvas по сравнению с SVG:
аппаратное ускорение всех построений (см. chrome://gpu/), т.к. обычно Canvas находится на отдельном GPU-слое;
возможность отрисовки неограниченного числа примитивов (SVG строит DOM, поэтому количество примитивов ограничено);
возможно получения контента из нефайловых источников (видео, вебкамера);
возможна произвольная попиксельная обработка изображения.

Недостатки Canvas по сравнению с SVG:
сложность разработки (из-за низкоуровневости функций);
невозможность изменить нарисованный «объект» (надо всё перерисовывать), отсюда следует сложность анимации построений;
неинтерактивность (например, события мыши будет принимать весь элемент Canvas);
невозможность автоматического учёта при построениях плотности пикселей экрана (плотность пикселей всегда явно предзадана через атрибуты width/height и стилевые свойства width/height);
нарисованные тексты нельзя выделить мышью, найти поиском на странице;
ни нарисованные тексты, ни всё изображение не могут получить поисковики (они не исполняют JavaScript).
